## Project setup

```bash
$ npm install
$ cp template.env .env
# Update the .env file with your own settings
$ npx prisma db push
$ npx prisma generate
$ npm run dev
# Open http://localhost:3000
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
